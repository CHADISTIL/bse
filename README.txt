📁 الملفات المطلوبة (يجب رفعها كلها معاً):
───────────────────────────────────────────
  BCElectric.html   ← الملف الرئيسي
  sw.js             ← Service Worker (إشعارات الخلفية)
  manifest.json     ← معلومات التطبيق
  icon-72.png       ← أيقونة
  icon-96.png       ← أيقونة
  icon-128.png      ← أيقونة
  icon-192.png      ← أيقونة (الرئيسية)
  icon-512.png      ← أيقونة (متجر)

🌐 طريقة النشر (GitHub Pages):
───────────────────────────────
1. ارفع كل الملفات في نفس مجلد الـ repository
2. فعّل GitHub Pages من Settings > Pages
3. افتح الرابط على الهاتف
4. اضغط "إضافة إلى الشاشة الرئيسية"

⚠️ مهم جداً:
───────────────
- يجب رفع كل الملفات في نفس مجلد واحد
- sw.js يجب أن يكون في نفس مستوى BCElectric.html
- HTTPS إلزامي للإشعارات (GitHub Pages يوفره تلقائياً)

✅ الإصلاحات في هذه النسخة:
───────────────────────────────
1. تحويل كامل إلى PWA قابل للتثبيت
2. إشعارات الهاتف تعمل حتى لو التطبيق في الخلفية
3. إصلاح منطق النقاط السالبة:
   - الرسالة تظهر فقط في أول الشهر
   - تحسب نقاط الشهر الماضي فقط (ليس الحالي)
   - تنبيه تحذيري قبل 3 أيام من نهاية الشهر
4. تغيير التسمية إلى BC Electric
